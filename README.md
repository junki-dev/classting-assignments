news-feed-api
==============

## Environments
- Node(v20.11.1), TypesScript
- Nest.js
- Yarn
- Jest
- MongoDB
- Docker, Docker-compose
- OpenAPI

## Getting Start

### Running Applications
```bash
$ docker-compose up -d
```

### Stopping Applications
```bash
$ docker-compose down
```

### Testing
```bash
$ yarn test:e2e

$ yarn test:e2e-cov
```
